# gdrb-spatial-analysis
This project explores regional economic disparities across Indonesia using spatial econometric models (SAR, SEM, SARMA) on 2023 PDRB data.

The analysis incorporates Spatial Autoregressive (SAR), Spatial Error Model (SEM), and Spatial Autoregressive Moving Average (SARMA) to account for spatial dependencies across provinces. Results indicate significant spatial autocorrelation (Moran’s I = 0.4993), with clear clustering patterns between high-growth regions (Java) and low-growth regions (Eastern Indonesia).

Model evaluation shows SARMA as the best-performing model, with high R² and low prediction error (RMSE, MAPE, AIC). Key influencing factors include population, HDI, labor participation, electricity access, and digital connectivity.

This project demonstrates the application of spatial analytics in understanding economic structure and supporting data-driven regional development policies.
